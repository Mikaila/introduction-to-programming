Windows Setup

Install Ruby

The easiest way to install Ruby on Windows is the RubyInstaller. From the downloads page, select Ruby 1.9.2-p180 and follow the installation instructions to get Ruby setup on your computer.

Install Git

Install a Text Editor

Any text editor will work but we recommend installing something that has some helpful programming features. A common editor you could install would be Notepad ++. It's free and easy to install. Just use the installer link from the Downloads page (or click here) and follow the instructions.

Install Ruby

Good news! Macs come pre-installed with ruby. You will, however, need git and a good text editor.

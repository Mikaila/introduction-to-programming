Windows Setup

Install Ruby

The easiest way to install Ruby is to go ahead and install the RailsInstaller. Click on the big green button and you are off and running. This will install Ruby as well as a few other tools that you will find useful in using Ruby moving forward (especially if you are interested in seeing how to develop web applications). This will also install Git, a version control system, we will be using extensively with Github (don't worry, we will explain all of this and why it is important as we go along).

  • Ruby: The Programming Language we will be using
  • Git: The version control system we will be using
  • Rails: The tools to build web applications with Ruby
  • Sqlite: A simple and lightweight database
  • DevKit: General tools that Ruby needs to build libraries for you

Install a Text Editor

When programming, you will need something to enter your code into. A text editor is usually best. This is an application that lets you enter text, but does not do ANY formatting of any kind (like word would do). Many editors have features that make your programming life a little easier (like highlighting certain words, or letting you create buttons that run certain bits of code).

One common editor you could install would be Notepad ++. It's free and easy to install. Just use the installer link from the Downloads page (or click here) and follow the instructions.

Another common editor that will do the trick, and is also free, is Crimson Editor. It's a fun little editor that has some great configuration options (including changing your color scheme). You can download it here. You can download the latest build here.

Install Ruby

Good news! Macs come pre-installed with ruby. You will, however, need git and a good text editor.

Install Git

You will need to install Git, a version control system, we will be using extensively with Github (don't worry, we will explain all of this and why it is important as we go along). In order to install git, simply go here and start the download for git. Install the .pkg file for git.

Next, follow these instructions on setting up your ssh keys. These are the keys that allow you to work with github. Again, just follow the instructions, and do not worry, we will explain everything in class.

After you have installed TextMate, simply unzip this file and double click the Vibrant Ink theme. It will install it for you automagically.
